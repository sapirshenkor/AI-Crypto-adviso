# Frontend Skill

Stack:
- React
- Vite
- TypeScript

Structure:
- src/pages/
- src/components/
- src/services/
- src/hooks/
- src/types/
- src/config/

Rules:
- Use typed API responses.
- Keep API calls in service files.
- Handle loading states.
- Handle error states.
- Handle empty states.
- Keep components small and reusable.
- Do not call external APIs directly.
